# 🌍 Wanderly – Travel Website

A responsive travel website developed using **HTML** and **CSS**, designed with a clean and user-friendly interface. The website consists of five pages and is optimized for mobile devices.

## 🚀 Features
- Responsive design for mobile screens
- Clean and modern user interface
- Easy navigation across all pages
- Attractive travel-themed layout
- Newsletter subscription section
- Social media icons in the footer

## 📄 Pages
- Home
- About
- Explore
- Login
- Contact

## 🛠️ Technologies Used
- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

## 📱 Responsiveness
- Optimized for mobile screens using CSS Media Queries
